# Recommendation Backend

A backend service that provides a personalized, scrollable content feed (Inshorts-style) for movies and TV shows.

## Features

- Personalized recommendations
- Learns from user behavior (watch time, skips)
- Supports movies and TV shows
- Pagination for infinite scrolling
- Stateless and frontend-agnostic

## Tech Stack

- Node.js
- Express
- MongoDB
- TMDB API

## Setup Instructions

```bash
npm install
npm run dev
```
